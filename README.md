# pavelsolano.mx

Personal website of Pavel Solano García — Systems Scientist & Complex Systems Designer.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Setup

```bash
npm install
npm run dev
```

## Photo
Add your photo as `public/images/pavel-photo.jpg` (portrait, 480×600px recommended).

## Deploy
Push to GitHub → Vercel auto-deploys.
